## Deep Q-Learning

Well-tuned DQN for low dimensional control tasks.

#### Run Code

`$ python cartpole_dqn.py --device=cpu --episodes=150 --model_dir=cartpole-model` to run code.

`$ python cartpole_dqn.py -h` for help messages.
